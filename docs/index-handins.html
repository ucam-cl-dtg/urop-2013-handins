Handins Documentation

  • Creates a bin

    Adds a newly created bin to the database the with specified details.

    Parameters
    name (form parameter)
    The name of the newly created bin.
    owner (form parameter)
    The CrsId of the owner of the bin.
    Response
    Status 400
    Unacceptable name.
    Status 202
    Bin successfully created but no Dos found for the supervisor. Returns a map containing the ID and the name of the bin.
    Status 200
    Bin successfully creted. Returns a map containg the ID and the name of the bin.

  • View bin

    Asks for the details of the bin if permissions are allow it.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin not visible to the user and/or user cannot access the requested bin details.
    Status 200
    Returns a map containing the id, name, archive status and peer marking status of the bin.

  • Add submission

    Saves an uploaded submission.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    file name (form parameter)
    The name of the uploaded file.
    file byte array (form parameter)
    The byte array representing the uploaded file's content.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin not visible to the user and/or user cannot upload into respective bin.
    Status 500
    Unable to save the file or unable to manage the saved file.
    Status 200
    A map containing the id of the submission and the id of the bin.

  • Bin access permissions

    Asks for the list of bin access permissions if the queries is allowed to.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin not visible to the user and/or user cannot access the requested bin details.
    Status 200
    A map containing a list of strings representing the CrsIds of the users who have permissions to upload to the bin.

  • Bin questions

    Asks for the questions assigned to if user is allowed to.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin not visible to the user and/or user cannot access the requested bin details.
    Status 200
    A list of maps each containing the id and the name of a question from the bin.

  • Bin uploaded submissions by the user

    Resturns the list of submissions that have bin uploaded into the bin by the calling user.

    Parameters
    binid (path paramefater)
    The id of the queried bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin not visible to the user and/or user cannot access the requested bin details.
    Status 200
    A list of maps containing the index and the id of the submissions that the user has uploaded into the bin.

  • The pdf splitting screen

    Splits the specified submission from the bin according to a given list of parameters.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    submissionid (path parameter)
    The id of the queried submission.
    splitting form (form parameter)
    5 equally long lists: id[] (the id of the question), startpage[] and endpage[] (the starting and ending page for that question), startloc[] and endloc[] (floats between 0 and 1 specifying the exact procentual coordinate on pages for the question)
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin not visible to the user and/or user cannot access the requested bin details.
    Status 400
    Unacceptable split.
    Status 500
    Unable to save split files.
    Status 200
    Successfully saved.

  • Delete Bin

    Deletes the bin if there are no uploads associated to it.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin cannot be deleted. Either it does not exist or contains uploaded files. Error details are specified.
    Status 204
    Successfully deleted.

  • Updates the Bin

    Adds the provided user to the list of access permissions of the bin.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    binForm (form parameter)
    The id of the bin and any of the changeable bin details: owner, name, archive state, peermarking state.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin cannot be modified by the user.
    Status 204
    Successfully modified.

  • Add Access Permissions to Bin

    Adds the provided users to the list of access permissions of the bin.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    users[] (form parameter)
    A list of crsIds to be granted permission to access and upload into the bin.
    user (form parameter)
    A crsId to be granted permission to access and upload into the bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin permissions cannot be modified by the user.
    Status 202
    Permissions successfully added but some users don't have the Dos added.
    Status 204
    Permissions successfully added.

  • Delete Access Permissions from Bin

    Delete the specified users from the bins access permission list.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    users[] (path parameter)
    The list of crsIds to be deleted from the list of permissions of the bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin permissions cannot be modified by the user.
    Status 204
    Permissions successfully deleted.

  • Delete Access Permission from Bin

    Delete the specified user from the bins access permission list.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    user (path parameter)
    The crsId to be deleted from the list of permissions of the bin.
    Response
    Status 404

    Status 403
    Bin permissions cannot be modified by the user.
    Status 204
    Permissions successfully deleted.

  • Add Marking Permissions to Bin

    Adds the provided relations to the list of marking permissions of the bin.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    markingUsers[] (form parameter)
    A list of crsIds to be granted permission to access mark the specified answers from the bin.
    questionIds[] (form parameter)
    A list of question ids whose answers from certain users will become available for marking by other users.
    questionOwners[] (form parameter)
    A list of crsIds whose answers to the specified questions will become available to the marking users.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin permissions cannot be modified by the user.
    Status 204
    Permissions successfully added.

  • Delete Marking Permissions from Bin

    Deletes the specified relations from the bins marking permission list.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    markingUsers[] (form parameter)
    A list of crsIds to be disallowed from marking the specified answers from the bin.
    questionIds[] (form parameter)
    A list of question ids whose answers from certain users will become unavailable for marking by other users.
    questionOwners[] (form parameter)
    A list of crsIds whose answers to the specified questions will become unavailable to the marking users.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin permissions cannot be modified by the user.
    Status 204
    Permissions successfully deleted.

  • Add Questions to Bin

    Adds the provided questions to the bin.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    name[] (form parameter)
    A list of question name to be added to the bin.
    link[] (form parameter)
    A list of links for the specified questions.
    name (form parameter)
    The name of a question to be added to the bin.
    link (form parameter)
    The link for the question.
    Response
    Status 404
    Bin does not exist.
    Status 400
    Invalid question name provided.
    Status 403
    Bin questions cannot be modified by the user.
    Status 204
    Questions successfully added.

  • Delete Access Permissions from Bin

    Delete the specified users from the bins access permission list.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    questionsId[] (query parameter)
    The list of crsIds to be deleted from the list of permissions of the bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin questions cannot be modified by the user.
    Status 200
    Unanswered questions have been successfully deleted and returns a list of undeletable questions.
    Status 204
    All questions successfully deleted.

  • Delete Access Permission from Bin

    Delete the specified user from the bins access permission list.

    Parameters
    binid (path parameter)
    The id of the queried bin.
    questionid (path parameter)
    The id of the question to be deleted from the bin.
    Response
    Status 404
    Bin does not exist.
    Status 403
    Bin questions cannot be modified by the user.
    Status 200
    Question was not deleted because it was answered.
    Status 204
    Question successfully deleted.

    Parameters

    questionsetid (form parameter)
    The id of the questionSet to be improted.
    Response
    Status 404
    Bin does not exist.
    Status 400
    Invalid question name provided.
    Status 403
    Bin questions cannot be modified by the user.
    Status 204
    Questions successfully added.
